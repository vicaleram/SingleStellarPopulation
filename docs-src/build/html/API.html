class SingleStellarP.SingleStellarP(M_1, M_2, alpha, epsilon_0)¶

Class containing the operations and graphs for the SingleStellarPopulation mododule

Variables

  • self.M_1 – Lower edge of initial mass function

  • self.M_2 – Upper edge of initial mass function

  • alpha – Power index used for the IMF

  • epsilon_0 – parameter value for the number of stars

SingleStellarP.get_B_pow(N, model_B_array=array([0, 1, 5, 10, 20]), Bmin=0.01, Bmax=30.0, alphaB=0.8)¶

Function that generates a power law disribution of initial magnetic field functions

Parameters

  • N – Number of stars in the cluster

  • model_B_array – The model_B_array represent the lower edge of the magnetic field bins in kG.

  • Bmin – Bmax is the lower edge of the last bin

  • Bmax – Bmax is the higher edge of the last bin

  • alpha – Value for the powerlaw index
